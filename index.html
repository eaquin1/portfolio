Creations

Energy Guide

An interactive calculator for a user to make informed decisions on their electricity usage, built with Python/Flask, and JavaScript. It shows the user how clean their electricity grid is at the current time, using the Watt Time API, and how much it costs to use various home appliances.

Dexcom Tracker

A web app using the Dexcom API and the Edamam API to visualize changes in glucose levels. The Dexcom API has blood glucose levels that are measured every five minutes, and the Edamam API has carbohydrate information for meals. The web app combines a user's meal information with their blood sugars over a range of dates, to allow for a better understanding of the user's blood sugar fluctuations. The app uses React, Express.js, PostgreSQL, Redis and Passport.js.
